# HttpClient Quick Start

## 前提

以`HttpClient 4.5`为例，要求Java 1.5+。

## 环境准备

以maven为例，引入所需依赖包

```xml
<dependency>
  <groupId>org.apache.httpcomponents</groupId>
  <artifactId>httpclient</artifactId>
  <version>4.5.4</version>
</dependency>
```

```xml
<dependency>
  <groupId>org.apache.httpcomponents</groupId>
  <artifactId>fluent-hc</artifactId>
  <version>4.5.4</version>
</dependency>
```
