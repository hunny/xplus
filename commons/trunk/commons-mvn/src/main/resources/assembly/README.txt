 客户端程序使用说明
=============================================

请按照以下步骤操作：

Step 1. 安装与配置JRE

确保已经正确安装JRE或JDK 1.7以上版本，并且已经正确设置环境变量JAVA_HOME。

Step 2. 下载安装包

下载安装包application-x.x.zip，并解压缩。

Step 3. 配置

打开applicationContext.properties，配置对应的参数。

Step 4. 首次运行

在命令行提示符下，进入解压缩目录，输入如下命令以即可查看使用帮助：
  run help -h
